### commit规范

| 类型         | 详细介绍 |
| ----------- | ----------- |
| feat      | 新功能、新特性   |
| fix	bugfix   | 修改问题        |
|refactor	|代码重构|
|docs	|文档修改|
|style	|代码格式修改，注意不是css修改|
|test	|测试用例修改|
|chore	|其他修改，比如构建，依赖管理|
