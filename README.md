# git常用命令
| 功能              | 指令                            |
| -----------------| ------------------------------  |
| 初始化git仓库     | `git init`                       |
|  设置用户名       | `git config --global user.name`+`你的名字` |
|  查看用户名         | `git config --global user.name` |
|  设置邮箱       | `git config --global user.email`+`邮箱名` |
|  查看邮箱       | `git config --global user.email` |
|  添加所有文件       | `git add .` |

# commit规范

| 类型         | 详细介绍 |
| ----------- | ----------- |
| feat      | 新功能、新特性   |
| fix	bugfix   | 修改问题        |
|refactor	|代码重构|
|docs	|文档修改|
|style	|代码格式修改，注意不是css修改|
|test	|测试用例修改|
|chore	|其他修改，比如构建，依赖管理|
